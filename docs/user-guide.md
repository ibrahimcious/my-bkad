# Panduan Pengguna — Dashboard BKAD Pasuruan

Dashboard ini menyajikan ringkasan realisasi anggaran (LRA) Badan Keuangan
dan Aset Daerah Kabupaten Pasuruan dalam satu halaman. Dashboard bersifat
**hanya-baca**: data tidak dapat diubah dari sini, perbaikan dilakukan pada
sistem sumber (SIPD Penatausahaan).

## Masuk ke Aplikasi

1. Buka alamat dashboard pada peramban (browser) di komputer Anda.
2. Masukkan **email** dan **kata sandi** yang telah diberikan.
3. Klik **Masuk**.

Jika email atau kata sandi salah, akan muncul pesan
"Email atau kata sandi salah." Pastikan penulisan sudah benar.

## Membaca Dashboard

Setelah masuk, halaman utama menampilkan:

- **Tiga kartu ringkasan** di bagian atas:
  - *Total Anggaran* — total pagu anggaran.
  - *Total Realisasi* — total anggaran yang telah terealisasi.
  - *Persentase Serapan* — perbandingan realisasi terhadap anggaran.
- **Grafik Kelompok Belanja** — perbandingan anggaran dan realisasi untuk
  Belanja Operasi, Modal, Tak Terduga, dan Transfer.
- **Serapan Tertinggi dan Terendah** — program dengan persentase serapan
  paling tinggi dan paling rendah.
- **Tabel Program** — daftar program dengan anggaran terbesar. Kolom
  Anggaran, Realisasi, dan Serapan dapat diurutkan dengan menekan judul
  kolomnya.

Keterangan **"Terakhir diperbarui"** di bawah judul menunjukkan tanggal
data terakhir diunggah. Bila tertulis bahwa belum ada data, berarti berkas
LRA belum pernah diunggah.

## Memperbarui Data (khusus petugas unggah)

Hanya akun petugas unggah (uploader) yang dapat memperbarui data:

1. Buka menu **Administrasi / Unggah Data** (`/admin/upload`).
2. Pilih berkas Excel LRA hasil ekspor dari SIPD Penatausahaan.
3. Klik **Unggah**.
4. Tunggu hingga muncul pesan keberhasilan beserta jumlah baris data.

Setiap unggahan **menggantikan seluruh data sebelumnya**. Riwayat unggahan
ditampilkan pada tabel di halaman yang sama.

Akun Kepala dan Sekretaris tidak memiliki akses ke halaman ini.

## Keluar

Klik tautan **Keluar** di pojok kanan atas untuk mengakhiri sesi. Demi
keamanan, selalu keluar setelah selesai, terutama pada komputer bersama.

## Bantuan

Bila dashboard tidak dapat diakses, data terlihat tidak wajar, atau Anda
lupa kata sandi, hubungi petugas pengelola aplikasi di Sekretariat BKAD.
